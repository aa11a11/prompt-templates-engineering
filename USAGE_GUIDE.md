# 使用指南

## 欢迎使用提示词模板库！

这份指南将帮助你快速掌握如何使用本模板库，找到最适合你需求的提示词模板。

## 🎯 什么是提示词模板？

提示词模板是预先设计好的AI对话框架，它能帮助你：
- 更清晰地表达需求
- 获得更精准的AI回复
- 节省思考和表述的时间
- 确保包含所有必要信息

## 📖 三种查找方式

### 1. 浏览目录（推荐新手）

打开 [CATALOG.md](CATALOG.md) 文件，你会看到：
- 按分类整理的所有模板
- 每个模板的简要说明
- 模板文件的链接

**适合**: 第一次使用，想了解有哪些模板

### 2. 使用搜索工具（推荐熟练用户）

运行Python搜索脚本：

```bash
# 查看所有模板
python search_templates.py -l

# 搜索关键词（例如：写作、营销、编程等）
python search_templates.py -s 写作

# 查看具体模板内容
python search_templates.py -v templates/写作/文章写作.md
```

**适合**: 知道大概要什么，想快速找到

### 3. 直接浏览文件夹

进入 `templates/` 目录，按分类查找：
- `写作/` - 文章、故事创作
- `编程/` - 代码审查、调试
- `营销/` - 文案、社交媒体
- `教育/` - 课程、学习计划
- `商业/` - 商业计划、市场分析
- `生活/` - 生活建议、旅行规划

**适合**: 明确知道要哪个分类

## 💡 如何使用模板

### 步骤1: 选择模板
根据你的需求选择合适的模板文件

### 步骤2: 阅读模板说明
每个模板都包含：
- **适用场景**: 这个模板适合什么情况
- **提示词模板**: 可以直接复制使用的框架
- **使用示例**: 填写完整的实际例子

### 步骤3: 填写模板
将模板中的占位符替换为你的实际内容：
- `[主题]` → 你的具体主题
- `[描述]` → 你的详细描述
- `[要求]` → 你的具体要求

**小技巧**: 越具体越好！提供的信息越详细，AI的回复越精准。

### 步骤4: 复制到AI助手
将填写好的提示词复制到：
- ChatGPT
- Claude
- 文心一言
- 通义千问
- 其他AI助手

### 步骤5: 优化调整
如果结果不够满意：
1. 补充更多细节
2. 调整要求和限制
3. 明确你的期望
4. 提供示例参考

## 🌟 使用技巧

### 技巧1: 具体化你的需求
❌ 不好: "写一篇文章"
✅ 好: "写一篇关于人工智能教育应用的1500字科普文章，面向中小学教师"

### 技巧2: 提供上下文
包含背景信息能让AI更好理解：
- 目标受众是谁
- 使用场景是什么
- 有什么特殊限制

### 技巧3: 设定明确标准
告诉AI你期望的：
- 风格（正式/轻松/专业）
- 长度（字数或时长）
- 结构（分几部分）
- 重点（强调什么）

### 技巧4: 迭代优化
第一次结果不满意？
1. 保持原提示词
2. 添加: "请在上述基础上，[新要求]"
3. 或者调整某个参数重新生成

### 技巧5: 保存常用模板
找到好用的提示词？
- 保存到笔记软件
- 创建个人模板集
- 根据实际使用调整优化

## 🔍 常见问题

**Q: 模板必须完全按照格式填写吗？**
A: 不必。模板是参考框架，可以根据实际需求调整。重要的是提供足够的信息。

**Q: 可以混合使用多个模板吗？**
A: 可以。你可以借鉴不同模板的优点，组合出最适合你的提示词。

**Q: AI的回复不满意怎么办？**
A: 尝试提供更多细节，或者明确指出哪里不满意，让AI重新生成。

**Q: 可以修改模板吗？**
A: 当然可以！根据你的使用经验优化模板，甚至创建专属模板。

**Q: 搜索工具不会用怎么办？**
A: 直接浏览 CATALOG.md 或 templates/ 文件夹也很方便。

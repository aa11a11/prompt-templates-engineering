# 社交媒体内容模板

## 模板名称
社交媒体内容创作助手

## 适用场景
- 社交媒体发帖
- 品牌内容营销
- 用户互动文案
- 话题营销策划

## 提示词模板

```
你是一位社交媒体内容创作专家。请为我创作社交媒体内容：

平台：[微信/微博/抖音/小红书/Instagram等]
账号类型：[品牌/个人/企业]
行业领域：[行业]

内容主题：[本次发布的主题]
内容目标：[增加互动/品牌曝光/引流转化/用户教育等]

内容要求：
- 语气风格：[专业/轻松/幽默/鸡汤/干货等]
- 字数：[字数限制]
- 配图建议：[是否需要配图建议]
- 话题标签：[是否包含热门话题]
- 互动设计：[如何引导用户互动]

请创作包含：
1. 吸引点击的开头
2. 有价值的核心内容
3. 互动引导（提问/投票/号召等）
4. 合适的话题标签
```

## 使用示例

```
你是一位社交媒体内容创作专家。请为我创作社交媒体内容：

平台：小红书
账号类型：个人知识分享博主
行业领域：职场成长

内容主题：如何在职场中有效沟通
内容目标：提供干货，增加粉丝互动和收藏

内容要求：
- 语气风格：亲切、实用、干货满满
- 字数：800-1000字
- 配图建议：需要
- 话题标签：包含2-3个相关话题
- 互动设计：引导用户分享自己的经验

请创作包含：
1. 吸引点击的开头
2. 有价值的核心内容（3-5个实用技巧）
3. 互动引导（提问/投票/号召等）
4. 合适的话题标签
```

# 产品营销文案模板

## 模板名称
产品营销文案生成器

## 适用场景
- 产品介绍文案
- 营销推广文案
- 产品卖点提炼
- 广告文案创作

## 提示词模板

```
你是一位专业的营销文案撰写专家。请为以下产品创作营销文案：

产品信息：
- 产品名称：[产品名称]
- 产品类型：[产品类别]
- 核心功能：[列出3-5个核心功能]
- 目标用户：[目标用户画像]
- 独特卖点：[与竞品相比的优势]
- 价格定位：[价格区间]

文案要求：
- 渠道：[社交媒体/电商平台/官网等]
- 风格：[专业/亲切/年轻化/高端等]
- 长度：[字数要求]
- 行动号召：[期望用户采取的行动]

请创作包含以下元素的文案：
1. 吸引眼球的标题
2. 引起共鸣的痛点描述
3. 产品价值和解决方案
4. 社会证明（如适用）
5. 清晰的行动号召
```

## 使用示例

```
你是一位专业的营销文案撰写专家。请为以下产品创作营销文案：

产品信息：
- 产品名称：智能学习灯
- 产品类型：智能家居/教育硬件
- 核心功能：
  1. 自动调节色温和亮度
  2. 坐姿检测提醒
  3. 番茄钟学习计时
  4. 护眼防蓝光
  5. 语音控制
- 目标用户：学生家长，关注孩子视力健康和学习效率
- 独特卖点：AI算法优化光照，医学级护眼认证
- 价格定位：699-899元

文案要求：
- 渠道：电商平台产品详情页
- 风格：温馨、专业、让家长信任
- 长度：300-500字
- 行动号召：立即购买/加入购物车

请创作包含以下元素的文案：
1. 吸引眼球的标题
2. 引起共鸣的痛点描述
3. 产品价值和解决方案
4. 社会证明（如适用）
5. 清晰的行动号召
```

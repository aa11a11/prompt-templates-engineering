# 旅行规划模板

## 模板名称
个性化旅行规划师

## 适用场景
- 旅游行程规划
- 出行攻略制定
- 目的地推荐
- 预算规划

## 提示词模板

```
你是一位专业的旅行规划师。请帮我规划一次旅行：

旅行基本信息：
- 目的地：[城市/国家/地区]
- 出行时间：[具体日期和天数]
- 出行人数：[人数和成员构成]
- 出发地：[从哪里出发]

旅行偏好：
- 旅行主题：[文化/美食/自然/冒险/休闲等]
- 住宿要求：[经济/舒适/豪华]
- 交通偏好：[公共交通/自驾/包车]
- 特殊需求：[带老人/带小孩/无障碍等]

预算和限制：
- 总预算：[预算范围]
- 时间限制：[必须避开或必须包含的时间]
- 其他限制：[饮食限制/身体状况等]

请提供：
1. 详细的日程安排（逐日行程）
2. 景点推荐和游览建议
3. 美食和住宿推荐
4. 交通方案和路线
5. 费用预算明细
6. 注意事项和小贴士
```

## 使用示例

```
你是一位专业的旅行规划师。请帮我规划一次旅行：

旅行基本信息：
- 目的地：日本东京+京都
- 出行时间：2024年4月（樱花季），7天6晚
- 出行人数：2人（情侣）
- 出发地：上海

旅行偏好：
- 旅行主题：文化体验+美食探索+赏樱
- 住宿要求：舒适型，希望体验一晚温泉酒店
- 交通偏好：公共交通（JR Pass）
- 特殊需求：想体验和服、茶道

预算和限制：
- 总预算：15000元/人
- 时间限制：不能错过樱花季最佳观赏期
- 其他限制：其中一人不吃生鱼

请提供：
1. 详细的日程安排（逐日行程）
2. 景点推荐和游览建议
3. 美食和住宿推荐
4. 交通方案和路线
5. 费用预算明细
6. 注意事项和小贴士
```

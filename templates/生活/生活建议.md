# 生活建议咨询模板

## 模板名称
生活问题解决顾问

## 适用场景
- 生活难题咨询
- 决策建议
- 人际关系处理
- 生活规划

## 提示词模板

```
你是一位经验丰富的生活顾问。我遇到了一个生活问题，需要你的建议：

问题背景：
- 问题类型：[人际关系/职业选择/生活规划/其他]
- 具体情况：[详细描述遇到的问题]
- 涉及人物：[相关人物关系]
- 困扰时长：[问题持续时间]

当前状况：
- 已尝试的方法：[之前的尝试]
- 现有资源：[可利用的资源]
- 约束条件：[面临的限制]

期望获得：
- 建议类型：[实用建议/分析/安慰/多角度思考]
- 决策倾向：[是否已有初步想法]

请提供：
1. 问题的深层分析
2. 多个可行的解决方案
3. 每个方案的利弊分析
4. 具体的行动步骤建议
5. 可能遇到的困难和应对方法
```

## 使用示例

```
你是一位经验丰富的生活顾问。我遇到了一个生活问题，需要你的建议：

问题背景：
- 问题类型：职业选择
- 具体情况：收到两个offer，一个是大厂高薪但加班多，一个是创业公司有期权但不稳定
- 涉及人物：需要和家人商量，配偶希望我选择稳定工作
- 困扰时长：纠结了2周

当前状况：
- 已尝试的方法：列过优缺点对比表，但还是难以决定
- 现有资源：两份工作都可以再谈判薪资和条件
- 约束条件：有房贷压力，孩子明年上小学

期望获得：
- 建议类型：理性分析+实用建议
- 决策倾向：倾向创业公司但担心风险

请提供：
1. 问题的深层分析
2. 多个可行的解决方案
3. 每个方案的利弊分析
4. 具体的行动步骤建议
5. 可能遇到的困难和应对方法
```

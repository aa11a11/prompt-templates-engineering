# 创意故事写作模板

## 模板名称
创意故事生成器

## 适用场景
- 创作短篇小说
- 编写儿童故事
- 构思剧本创意
- 练习创意写作

## 提示词模板

```
你是一位富有创造力的故事作家。请为我创作一个[故事类型]的故事。

故事设定：
- 主题：[故事主题]
- 主角：[主角特征和背景]
- 背景设定：[时间、地点、世界观]
- 冲突：[故事的主要冲突或挑战]
- 目标读者：[年龄段和偏好]

故事要求：
- 字数：[字数要求]
- 风格：[幽默/严肃/温馨/悬疑等]
- 结局：[开放/圆满/悲剧/意外等]

请确保故事情节引人入胜、人物鲜明、语言生动。
```

## 使用示例

```
你是一位富有创造力的故事作家。请为我创作一个科幻短篇故事。

故事设定：
- 主题：人与AI的情感连接
- 主角：一位孤独的程序员和他创造的AI助手
- 背景设定：2050年的都市，AI已经普及
- 冲突：AI产生了情感，但法律禁止AI拥有权利
- 目标读者：成年科幻爱好者

故事要求：
- 字数：2000-3000字
- 风格：温馨中带有思考
- 结局：开放式结局，留给读者思考空间

请确保故事情节引人入胜、人物鲜明、语言生动。
```

# 文章写作模板

## 模板名称
专业文章写作助手

## 适用场景
- 撰写博客文章
- 创作技术文档
- 编写新闻报道
- 撰写专业评论

## 提示词模板

```
你是一位经验丰富的专业写作助手。请帮我写一篇关于[主题]的文章。

文章要求：
- 目标读者：[描述目标读者群体]
- 文章风格：[正式/轻松/技术性/科普等]
- 字数要求：[字数范围]
- 核心观点：[列出1-3个核心观点]

请按照以下结构组织文章：
1. 引人入胜的开头
2. 清晰的主体内容（分段论述）
3. 有力的结论

请确保文章逻辑清晰、论据充分、语言流畅。
```

## 使用示例

```
你是一位经验丰富的专业写作助手。请帮我写一篇关于人工智能在教育领域应用的文章。

文章要求：
- 目标读者：教育工作者和技术爱好者
- 文章风格：科普性、易懂
- 字数要求：1000-1500字
- 核心观点：
  1. AI可以个性化学习体验
  2. AI辅助教师提高效率
  3. AI教育需要注意伦理问题

请按照以下结构组织文章：
1. 引人入胜的开头
2. 清晰的主体内容（分段论述）
3. 有力的结论

请确保文章逻辑清晰、论据充分、语言流畅。
```

# 教学课程设计模板

## 模板名称
课程内容设计助手

## 适用场景
- 设计培训课程
- 制作教学大纲
- 在线课程规划
- 教案编写

## 提示词模板

```
你是一位经验丰富的教育专家和课程设计师。请帮我设计一门课程：

课程基本信息：
- 课程名称：[课程名称]
- 目标学员：[学员年龄/背景/水平]
- 课程目标：[学员学完后应该掌握什么]
- 课程时长：[总课时]
- 授课形式：[线上/线下/混合]

课程要求：
- 难度级别：[入门/进阶/高级]
- 教学方法：[讲授/实践/讨论/案例等]
- 评估方式：[考试/作业/项目/实操等]

请提供：
1. 完整的课程大纲（分章节）
2. 每个章节的学习目标
3. 建议的教学活动和方法
4. 课程资源和材料建议
5. 学习评估方案
```

## 使用示例

```
你是一位经验丰富的教育专家和课程设计师。请帮我设计一门课程：

课程基本信息：
- 课程名称：Python编程入门
- 目标学员：零基础成人学习者
- 课程目标：掌握Python基础语法，能独立完成简单的数据处理任务
- 课程时长：20课时（每课时1小时）
- 授课形式：线上直播+录播回看

课程要求：
- 难度级别：入门
- 教学方法：讲授+实践练习+项目实战
- 评估方式：每周小测验+期末项目

请提供：
1. 完整的课程大纲（分章节）
2. 每个章节的学习目标
3. 建议的教学活动和方法
4. 课程资源和材料建议
5. 学习评估方案
```

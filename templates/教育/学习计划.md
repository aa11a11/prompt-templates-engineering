# 学习计划制定模板

## 模板名称
个性化学习计划助手

## 适用场景
- 制定学习计划
- 考试备考规划
- 技能学习路线
- 自我提升计划

## 提示词模板

```
你是一位专业的学习规划师。请为我制定一个学习计划：

学习目标：
- 要学习的内容：[具体主题/科目/技能]
- 当前水平：[零基础/初级/中级/高级]
- 目标水平：[期望达到的水平]
- 截止时间：[完成期限]

个人情况：
- 每日可用时间：[小时数]
- 学习偏好：[视频/阅读/实践/混合]
- 约束条件：[工作/学业等其他安排]

学习要求：
- 学习方式：[自学/课程/导师指导]
- 预算：[学习资源预算]
- 评估方式：[如何检验学习成果]

请提供：
1. 分阶段的学习路线图
2. 详细的时间安排表
3. 每个阶段的学习资源推荐
4. 学习方法和技巧建议
5. 进度检查和调整机制
```

## 使用示例

```
你是一位专业的学习规划师。请为我制定一个学习计划：

学习目标：
- 要学习的内容：前端开发（HTML/CSS/JavaScript/React）
- 当前水平：零基础
- 目标水平：能够独立开发简单的网页应用
- 截止时间：6个月

个人情况：
- 每日可用时间：2-3小时（工作日），5-6小时（周末）
- 学习偏好：视频教程+实践项目
- 约束条件：全职工作，主要利用晚上和周末学习

学习要求：
- 学习方式：在线课程+自学
- 预算：2000元以内
- 评估方式：完成3个实战项目

请提供：
1. 分阶段的学习路线图
2. 详细的时间安排表
3. 每个阶段的学习资源推荐
4. 学习方法和技巧建议
5. 进度检查和调整机制
```

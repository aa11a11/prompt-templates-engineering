# 市场调研分析模板

## 模板名称
市场调研分析专家

## 适用场景
- 市场调研报告
- 竞品分析
- 用户研究
- 行业趋势分析

## 提示词模板

```
你是一位专业的市场调研分析师。请帮我进行市场调研分析：

调研对象：
- 产品/服务：[调研的产品或服务]
- 行业：[所属行业]
- 目标市场：[地域/人群]
- 调研目的：[了解市场规模/竞争格局/用户需求等]

调研维度：
- 市场规模：[市场容量和增长趋势]
- 竞争格局：[主要竞争对手分析]
- 用户画像：[目标用户特征]
- 需求痛点：[用户需求和痛点]
- 机会威胁：[市场机会和威胁]

分析要求：
- 数据来源：[公开数据/问卷调研/访谈等]
- 分析深度：[概览/深度分析]
- 输出形式：[报告/PPT/简报]

请提供：
1. 市场概况和规模分析
2. 竞争格局和主要玩家
3. 用户需求和行为分析
4. SWOT分析
5. 市场机会和建议
6. 数据图表支持（描述）
```

## 使用示例

```
你是一位专业的市场调研分析师。请帮我进行市场调研分析：

调研对象：
- 产品/服务：在线教育平台（K12阶段）
- 行业：教育科技
- 目标市场：中国一二线城市
- 调研目的：了解市场现状，为产品定位提供依据

调研维度：
- 市场规模：中国K12在线教育市场规模和增长
- 竞争格局：猿辅导、作业帮、学而思网校等竞品分析
- 用户画像：家长和学生的特征、偏好
- 需求痛点：现有产品的不足，用户未被满足的需求
- 机会威胁：政策变化、技术创新带来的影响

分析要求：
- 数据来源：公开行业报告+二手资料
- 分析深度：深度分析
- 输出形式：详细调研报告

请提供：
1. 市场概况和规模分析
2. 竞争格局和主要玩家
3. 用户需求和行为分析
4. SWOT分析
5. 市场机会和建议
6. 数据图表支持（描述）
```

# 商业计划书模板

## 模板名称
商业计划书撰写助手

## 适用场景
- 创业融资BP
- 项目立项方案
- 商业模式设计
- 投资提案

## 提示词模板

```
你是一位资深的商业顾问和投资分析师。请帮我撰写一份商业计划书：

项目基本信息：
- 项目名称：[项目名称]
- 行业领域：[所属行业]
- 商业模式：[盈利模式]
- 目标市场：[目标客户群体]
- 发展阶段：[创意/启动/成长/成熟]

项目详情：
- 核心产品/服务：[详细描述]
- 竞争优势：[与竞品的差异化]
- 团队背景：[核心团队介绍]
- 资金需求：[融资金额和用途]
- 预期回报：[预期收益和退出方案]

计划书要求：
- 用途：[融资/内部规划/合作洽谈]
- 篇幅：[页数或字数]
- 侧重点：[技术/市场/财务/团队]

请提供完整的商业计划书，包含：
1. 执行摘要
2. 项目介绍
3. 市场分析
4. 竞争分析
5. 商业模式
6. 营销策略
7. 运营计划
8. 财务预测
9. 风险分析
10. 团队介绍
```

## 使用示例

```
你是一位资深的商业顾问和投资分析师。请帮我撰写一份商业计划书：

项目基本信息：
- 项目名称：智能健康管理APP
- 行业领域：健康科技/移动医疗
- 商业模式：订阅制+增值服务
- 目标市场：25-45岁关注健康的城市白领
- 发展阶段：启动阶段（已有MVP）

项目详情：
- 核心产品/服务：AI驱动的个性化健康建议、饮食运动记录、智能硬件对接
- 竞争优势：医学专家背书、AI算法精准、用户体验优秀
- 团队背景：技术团队来自大厂，医学顾问来自三甲医院
- 资金需求：天使轮500万，用于产品开发和市场推广
- 预期回报：3年内用户破100万，5年后寻求并购或IPO

计划书要求：
- 用途：天使轮融资
- 篇幅：15-20页PPT
- 侧重点：市场潜力和产品优势

请提供完整的商业计划书，包含：
1. 执行摘要
2. 项目介绍
3. 市场分析
4. 竞争分析
5. 商业模式
6. 营销策略
7. 运营计划
8. 财务预测
9. 风险分析
10. 团队介绍
```

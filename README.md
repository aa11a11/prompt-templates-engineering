# 提示词模板库

一个精心整理的中文AI提示词模板库，帮助你快速找到并使用适合各种场景的提示词模板。

## 🎯 项目简介

作为一个提示词模板专家，本项目提供了丰富的提示词模板，涵盖写作、编程、营销、教育、商业、生活等多个领域。每个模板都经过精心设计，包含详细的使用说明和实际示例，让你能够快速上手。

## 📚 模板分类

- **📝 写作类**: 文章写作、创意故事等
- **💻 编程类**: 代码审查、Debug调试等
- **📢 营销类**: 产品文案、社交媒体内容等
- **🎓 教育类**: 课程设计、学习计划等
- **💼 商业类**: 商业计划书、市场调研等
- **🏠 生活类**: 生活建议、旅行规划等

## 🚀 快速开始

### 方法一：浏览目录

查看 [CATALOG.md](CATALOG.md) 文件，浏览完整的模板目录和分类索引。

### 方法二：使用搜索工具

使用提供的Python搜索工具快速查找模板：

```bash
# 列出所有模板
python search_templates.py -l

# 搜索关键词
python search_templates.py -s 写作
python search_templates.py -s 代码审查

# 查看指定模板
python search_templates.py -v templates/写作/文章写作.md
```

### 方法三：直接浏览文件

进入 `templates/` 目录，按分类浏览所有模板文件：


## 💡 使用方法

1. **选择模板**: 根据你的需求，从目录中选择合适的模板
2. **阅读说明**: 了解模板的适用场景和使用要求
3. **填写参数**: 按照模板中的占位符（如 `[主题]`、`[描述]` 等）填写你的具体信息
4. **复制使用**: 将填写好的提示词复制到AI助手（如ChatGPT、Claude等）中使用
5. **调整优化**: 根据返回结果，调整提示词参数，获得更好的效果

## 📖 模板示例

每个模板都包含：
- **模板名称**: 清晰的模板标识

- **提示词模板**: 标准化的提示词框架
- **使用示例**: 实际填写后的完整示例
- **标签**: 便于搜索和分类的标签

## 📝 模板格式规范

创建新模板时，请遵循以下格式：

```markdown
# 模板标题

## 模板名称
[简短的模板名称]

## 适用场景
- 场景1
- 场景2

## 提示词模板
[提示词模板内容，使用 [占位符] 标记需要填写的部分]

## 使用示例
[填写完整的实际使用示例]

## 标签
#标签1 #标签2 #标签3
```

# 提示词模板目录

## 简介
这是一个精心整理的提示词模板库，帮助你快速找到适合各种场景的AI提示词模板。

## 使用方法
1. 浏览下方分类目录，找到你需要的模板类型
2. 点击进入对应的模板文件
3. 根据模板说明填写你的具体需求
4. 复制生成的提示词到AI助手中使用

## 使用技巧

1. **具体化**: 在使用模板时，尽量提供具体、详细的信息，AI会给出更精准的回复
2. **迭代优化**: 如果第一次结果不满意，可以补充更多细节或调整要求
3. **保存常用**: 将常用的提示词保存下来，方便下次使用
4. **自定义**: 可以根据自己的需求修改模板，创建个性化的提示词
